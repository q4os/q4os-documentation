Q4OS on Raspberry Pi

administrator manual


1. Introduction

Q4OS Raspberry Pi port is a free operating system based on Raspbian distribution, optimized for the Raspberry Pi family hardware, the series of credit card-sized single-board ARM computers. Q4OS is now one of a few distributions offering the full desktop environment on the ARM architecture.


2. Installation

The easiest way to get the Q4OS RPi system is to buy preinstalled SD card in our e-shop, otherwise you will have to install Q4OS operating system image on your own SD card. You will need another computer with a SD card writer to install the image. Official Q4OS RPi images are available for download from the Downloads section of the Q4OS website. After downloading the .zip file, unzip it to get the .img image file for writing to your SD card. With the image file you need to use an image writing tool to install it on your SD card. See the guide for your system:

  • Windows
  • Linux
  • Mac OS

Predefined login credentials are as follows:

username: pi
password: raspberry

We would also recommend for all to come across the official Raspberry Pi documentation. If you want to operate your Pi headless, please follow a detailed external guide Install Q4OS on Raspberry Pi.

Enjoy your Pi and don't forget to join the Raspberry Pi community and Q4OS forum.
